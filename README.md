# Dotfiles

![1](https://github.com/LukaJankovic/Dotfiles/raw/master/2.png)

## Install

Run `./dotdrop.sh install`.

## Controlling terminal (URxvt) window

Alt + Left click = move

Alt + Right click, drag = resize

## Other 

Super + Enter = center

Super + T = new terminal
